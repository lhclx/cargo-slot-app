# CargoPlus 弹窗重写完成

## 任务目标
将海运订单页面弹窗重写为 CargoPlus 风格，完全匹配真实界面布局。

## 完成内容

### 1. 弹窗布局重构
- **全屏布局**（宽 96%，高 90vh）
- **三栏结构**：
  - 左侧：树形导航栏（200px）
  - 中间：字段网格区（自适应）
  - 右侧：汇总面板（280px）

### 2. 树形导航栏
- 可折叠树结构，节点包括：
  - 📋 订单详情（可展开）
    - 基本信息
    - 船运信息
    - 货物操作
  - 💰 费用清单（可展开）
    - 应收费用
    - 应付费用
    - 利润分析
  - 📦 商品信息
  - 📋 PO信息
  - 📄 提单信息
- 支持展开/收起动画
- 点击节点切换中间内容区

### 3. 中间字段网格区
- 根据**选中的树节点**显示不同内容
- 默认显示"基本信息"
- 字段按行排列，每行 6-7 个字段
- 字段带标签，必填项标*
- 字段有搜索/下拉按钮
- 底部工具栏（拖车、仓储、报关、保险、代垫复选框 + JOB、批量、订舱等按钮）

### 4. 右侧汇总面板
- 三个可折叠区块：
  - 📦 箱量信息（尺寸|箱型|数量|已装|单柜重）
  - 📦 商品信息（序号|商品名称|数量|单位）
  - 📋 PO信息（PO号|商品|数量）
- 每个区块有新增/复制/移除按钮
- 表格行可选中高亮

### 5. 新增 JavaScript 函数
- `toggleTreeNode(el)` - 展开/收起树节点
- `switchTreeNode(el)` - 切换树节点并更新内容区
- `renderMainContent(nodeId)` - 根据节点ID渲染中间内容
- `togglePanelSection(headerElement)` - 展开/收起右侧面板区块
- `selectPanelRow(row)` - 选中面板表格行

### 6. CSS 样式
- 树形导航样式：
  - 节点悬停：background: #e6f0ff
  - 节点选中：background: #2f5496, color: white
  - 展开/收起图标旋转动画
  - 子节点缩进 24px
- 面板样式：
  - 折叠/展开图标旋转
  - 表格行选中高亮 #d6e4ff
